# -Screenshot-AI-Solver
一个支持自定义API的解题工具，只要你的程序里有API，按下shift+G后就可以框选解题，在程序主页面里可以清晰直观的查看到自己的token用量/This is a problem-solving tool that supports custom API keys.as long as you have an API configured in the program, you can press Shift+G to select an area on the screen and get the answer. Your token usage can be viewed clearly and intuitively on the main page of the program.
